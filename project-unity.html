Project Unity:
NFP Site Rebuild

A static-site rebuild for Project Unity NFP — the foundation for a custom WordPress migration on SiteGround. Multi-page architecture covering Catalyst Academy cohorts, Y-PHIC programming, leadership and research, and an interactive U.S. presence map.

Client

Project Unity NFP

Stack

HTML / CSS / JS → WordPress

Hosting

SiteGround

Overview

A nonprofit site that actually reflects the work.

Project Unity needed a site that could keep up with how the org has grown — multiple programs, multi-year cohorts, an evolving research arm, and a donation flow that doesn't fight the user. The rebuild ships as a clean static foundation, scoped so the next step (custom WordPress theme on SiteGround) is a port, not a redo.

What's in the build

Interactive Map

U.S. presence map on the homepage, highlighting active program locations.

Catalyst Academy

Cohort-specific pages spanning the 2023–2025 program cycles.

Y-PHIC Hub

Dedicated section for the youth public-health initiative.

Zeffy Donations

Embedded donation flow with no platform fees passed to donors.

Leadership & Research

Bios, publications, and the org's research output.

WordPress-Ready

Markup and asset structure designed to map cleanly into a custom theme.
