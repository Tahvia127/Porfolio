PublicPath:
Public-Sector Job & Partner Hub

A jobs-and-partnerships platform with automated daily ingestion. Python sync scripts feed a Postgres backend; the frontend serves curated jobs, partner profiles, and resource pages for people looking to move into public-sector and mission-driven work.

Backend

Python · PostgreSQL

Frontend

HTML · CSS · JS

Automation

GitHub Actions

Overview

Public-sector jobs are everywhere. Finding them shouldn't be a scavenger hunt.

PublicPath consolidates public-sector and mission-driven job listings into one curated hub, with partner organizations surfaced alongside the roles they're hiring for. The pipeline is fully automated — daily syncs pull new listings, dedupe against the existing dataset, and generate a digest for subscribers. The frontend is intentionally simple so that the data, not the chrome, does the work.

The data pipeline

sync_jobs.py → fetch + normalize listings → Postgres
sync_jobs2.py → secondary sources + dedupe
generate_digest.py → daily summary email
GitHub Actions → cron-triggered runs, no server to babysit

What's on the site

  • Jobs — filterable listings with role, organization, and location.
  • Partners — profiles for the orgs whose roles get featured.
  • Resources — guides and links for people transitioning into the public sector.
